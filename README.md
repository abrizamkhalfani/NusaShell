# NusaShell

COMMAND LIST:

HELP    - tampilkan bantuan
CLEAR   - bersihkan layar
TIME    - lihat waktu
DATE    - lihat tanggal
ROOT    - masuk mode admin
EXIT    - keluar terminal

ROOT COMMAND:

SYSINFO - info sistem
DIR     - lihat file
BACK    - keluar root mode

Copyright (C) 2026 Abrizam

## USAGE

Type commands directly after the prompt:

username@NusaShell>

Example:
user@NusaShell> help

## ROOT MODE

Command:
root

Enter password to access administrator mode.

ROOT COMMANDS:
sysinfo       - show system information
colorhack     - change terminal color
shutdownsim   - simulate shutdown
rootoff       - exit root mode

Default Password:
NusaShell
